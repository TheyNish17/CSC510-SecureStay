1. Set Theory Formulations

  • Let R be the set of all hotel rooms
  • Let F be the set of all hotel floors
  • Let O ⊆ R be the set of occupied rooms
  • Let A = R - O be the set of available rooms
  • Let BR ⊆ R be the set of balcony rooms
  • Let SR ⊆ BR be the set of suite rooms (all suites are balcony rooms)
  • Let FR ⊆ R be the set of family rooms
  • Functional Mappings: Loc : R → F and Book : G → R

2. Propositional Logic States

Var Proposition / Derived From
P Guest identity is verified
V Book(g) ∈ A (Assigned room is currently vacant)
R Requested room satisfies type requirements (FR / BR / SR)
T payment(g) = success (Payment transaction cleared)
Z Booking approved and room committed

3. Logical Truth Rules

Identity Verification: ¬P → ¬Z
Room Availability: ¬V → ¬Z
Payment Processing: ¬T → ¬Z
Final Approval Condition: Z ↔ (P ∧ V ∧ R ∧ T)
